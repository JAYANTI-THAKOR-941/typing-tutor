
🖥️ React.js Typing Tutor – TechSkillHub

A modern, interactive Typing Tutor application built with React.js, designed to help users improve typing speed and accuracy while learning in a fun and engaging way. Part of the TechSkillHub platform.

🚀 Features

Real-time Typing Test – Type displayed text and get instant feedback.

Speed Calculation – Words per minute (WPM) automatically calculated.

Accuracy Tracking – Highlights errors as you type.

Timer – Countdown timer for typing challenges.

Score History – Keep track of best performances.

Responsive Design – Works smoothly on desktop and mobile.

Theme Support – Light and dark modes for user preference.


This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
